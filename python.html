Intro to Python

Updated over 4 months ago
🐍

Python is one of the most beginner-friendly programming languages...

Lesson 1: Getting Started

📌 What is Python?

Python is a high-level, interpreted language used in web apps, data science, AI, and more.

🖥️ First Program
print("Hello, World!")

Lesson 2: Variables & Data Types

🔢 Data Types
name = "Alice"
age = 25
is_active = True
📦 Assigning Variables
x = 10
y = 3.5
z = "Python"

Lesson 3: Strings

✍️ String Basics
greeting = "Hello"
print(greeting.upper())
print(greeting.lower())
🔗 String Concatenation
name = "Alice"
print("Hello " + name)

Lesson 4: Lists

📋 Creating Lists
fruits = ["apple", "banana", "cherry"]
🔄 Iterating
for fruit in fruits:
    print(fruit)

Lesson 5: Dictionaries

📒 Key-Value Pairs
person = {"name": "Alice", "age": 25}
print(person["name"])

Lesson 6: If Statements

🔀 Conditions
x = 10
if x > 5:
    print("Greater")
else:
    print("Smaller")

Lesson 7: Loops

🔁 For Loops
for i in range(5):
    print(i)
🔄 While Loops
count = 0
while count < 5:
    print(count)
    count += 1

Lesson 8: Functions

⚙️ Defining Functions
def greet(name):
    return "Hello " + name

Lesson 9: Error Handling

⚠️ Try & Except
try:
    print(10 / 0)
except ZeroDivisionError:
    print("Error: Division by zero")

Lesson 10: File Handling

📂 Writing Files
with open("data.txt", "w") as f:
    f.write("Hello File!")
📖 Reading Files
with open("data.txt", "r") as f:
    print(f.read())

Lesson 11: Classes & Objects

🏗️ Defining a Class
class Person:
    def __init__(self, name):
        self.name = name

p = Person("Alice")
print(p.name)

Lesson 12: Modules & Libraries

📦 Importing
import math
print(math.sqrt(16))
